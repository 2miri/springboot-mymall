# [프로젝트소개]

IntelliJ, SpringBoot등을 활용하여 동적 홈페이지를 구상해보자!



### 주요 기능
- 로그인 (제3자로그인 포함) / 로그아웃
- 회원가입
- 상품 상세보기
- 찜하기
- 찜 목록 보기

  

#### 개발 환경

JDK 1.8

IDE : IntelliJ
